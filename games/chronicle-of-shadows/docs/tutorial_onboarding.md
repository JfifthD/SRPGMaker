# Tutorial & Onboarding

SRPG 장르 특유의 높은 진입 장벽(AP 결제, CT 턴 관리, 방향 지정)을 완화하기 위해 초반 챕터를 튜토리얼로 활용합니다.

## 1. 단계별 학습 곡선 (Learning Curve)

| 스테이지      | 목표 (유저가 배울 점) | 시스템 개방 수준                                                  |
| ------------- | --------------------- | ----------------------------------------------------------------- |
| **Prologue**  | 맵 이동 및 기본 타격  | CT 게이지 + 단순 이동 및 공격 활성화                              |
| **Chapter 1** | AP 경제 활용법        | Max AP 5 개방, 'Wait' 커맨드로 AP 저장 및 스킬 한 방 활용법 교습  |
| **Chapter 2** | 방향성 및 배후 타격   | Facing 기능 개방, 몹의 측면/후면 공격 시 추가 피해 및 임팩트 체감 |
| **Chapter 3** | 지형 및 스킬 연계     | ZOC, Z-axis 고저차, 체인어택(Affinity) 및 특화 어빌리티 해금      |
| **Chapter 4** | 종합 응용             | 모든 기능 개방 - 보스전                                           |

## 2. 인터랙티브 가이드라인

단순 텍스트 설명보다, 유저가 특정 타일로 유닛을 옮겨야만 진행 가능한 강제 지시선(`ITooltipRenderer` 혹은 `HighlightTile`) 제공

- **첫 번째 적 타겟팅 시:** 공격 범위를 빨간색 타일 보더라인으로 깜빡여 유도.
- **최초 `End Turn` 클릭 시:** 어느 방향(N/E/S/W)을 바라보아야 안전한지 아이콘+화살표 피드백 추가.

## 3. UI 헬퍼 기능 (비숙련자 보조)

- **Undo 기능:** 기본 옵션 활성화(Max 5~10회 버퍼 노출), 언제든 되돌려서 "실수"를 시도(Trial & Error) 해볼 수 있게 배려
- **Danger Zone(위협 범위):** 단축키가 아니라 UI 스위치 버튼 형태로 배치하여, 유저가 언제라도 적의 최대 위협 반경을 직관적으로 파악할 수 있도록 도우미 역할
